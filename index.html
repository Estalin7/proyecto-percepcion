Qué hace esta demo

Detecta señas médicas en tiempo real, confirma palabras por persistencia, arma una frase clínica y puede reproducirla usando la voz del navegador.

Estado del prototipo

El flujo local ya es funcional. Las siguientes fases apuntan a reconocimiento continuo, integración con Spark, MLOps y una operación más formal del sistema.

Cómo usar Guía rápida

📏Ubícate a una distancia media para que se vean rostro, torso y manos.
